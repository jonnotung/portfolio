What's playing?

JQuery-based web app

https://jonnotung.github.io/whatsPlaying/

This is a web app made using HTML, CSS and Javascript/JQuery. Users input the era and genre of movie they want to see, and the app dynamically selects 3 actors who match these selections. Users then select one of these actors and a movie suggestion, along with synopsis and poster image, is displayed. Utilizes the The Movie Database API. Made in 1 week by paired-programming with @DarwinLyons during Hackeryou's web development immersive program.

Class database

React-based web app

https://jonnotung.github.io/JonathanTungProject5/

This is a web app made using the React framework, utilizing Firebase as a database. Users can add students and the classes they are enrolled in into a database. Students can be searched for and their enrollment information updated. A stats tab gives summaries of what's in the database: students and what classes they are in, and classes and which students are taking them. Made in 1 week during Hackeryou's web development immersive program.

The wise mist

React-based web game

https://jonnotung.github.io/JonathanTungProject5/

This is a web app made using the React framework. Users enter their name and a keyword at the start. They then navigate a maze with moving obstacles to reach the wise mist at the end, which will give a personalized piece of advice based on the keyword entered at the start. Utilizes the advice slip API. Made in 1 week with @derekmurr, @jonathanmcodes and @sooyshim during Hackeryou's web development immersive program.
