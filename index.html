What's Playing?

What's playing?

JQuery-based web app

Link Github repo

This is a movie suggestion web app where users input the era and genre of movie they want to see, and the app dynamically selects 3 actors who match these selections. Users then select one of these actors and a movie suggestion, along with synopsis and poster image, is displayed. Utilizes the The Movie Database API. Made in 1 week with Darwin Lyons during Hackeryou's web development immersive program.

Class Database

Class database

React-based web app

Link Github repo

This is a web app made using the React framework, utilizing Firebase as a database. Users can add students and the classes they are enrolled in into a database. Students can be searched for and their enrollment information updated. A stats tab gives summaries of what's in the database: students and what classes they are in, and classes and which students are taking them. Made in 1 week during Hackeryou's web development immersive program.

The Wise Mist

The wise mist

React-based web game

Link Github repo

This is a 2-d maze game where users enter their name and a keyword at the start. They then navigate a maze with moving obstacles to reach the wise mist at the end, which will give a personalized piece of advice based on the keyword entered. Utilizes the advice slip API. Made in 1 week with Jonathan McWilliam, Derek Murr, and Soo Yeon Shim during Hackeryou's web development immersive program.
